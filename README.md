# 360 V6 ImmortalWrt 定制固件说明

## 📝 版本背景
此前编译的版本为最原始的基础镜像，功能较为精简。在实际日常使用中，发现部分基础需求无法便捷满足。为提升实用性，同时兼顾设备性能，特此在原版基础上进行轻量化增强。

## 🧩 集成插件清单
仅精选并集成以下 5 款高频实用插件，满足日常核心需求：
- `luci-app-opkg`：增强型软件包管理界面，支持可视化搜索与安装
- `luci-app-fileassistant`：轻量级文件管理助手，方便配置备份与文件传输
- `luci-app-wol`：网络唤醒（Wake-on-LAN），支持远程开机内网设备
- `luci-app-autoreboot`：定时自动重启，保持系统长期稳定运行
- `luci-app-adguardhome`：高性能 DNS 过滤与广告拦截/家长控制

## 🎯 设计理念
**「够用就好，稳定优先」**
鉴于 360 V6（高通 IPQ60xx 平台）的硬件规格与存储限制，本固件**不再考虑堆砌额外功能**。所有集成插件均经过严格筛选，力求在功能性、系统流畅度与资源占用之间取得最佳平衡。

## 📌 版本定位
- 本版本为 **功能定型版**，后续仅跟进官方内核与安全更新，不再新增任何第三方插件。
- 固件已针对 360 V6 硬件特性进行基础适配，开箱即用。
- 建议保持轻量使用习惯，避免安装重型服务（如 Samba、Docker、复杂代理等），以免影响无线稳定性与系统响应。

## ⚠️ 刷机提示
1. 刷机前请务必通过「系统 → 备份/升级」导出当前配置。
2. 首次登录 LuCI 建议修改默认密码，并按提示完成 AdGuardHome 初始化。
3. 如遇网络异常或插件冲突，可尝试恢复出厂设置或重新刷入完整固件。

> 精简，是为了更持久地稳定运行。感谢使用。另外这个luci-app-samba4要不要安装。
