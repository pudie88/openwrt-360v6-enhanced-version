# 📦 ImmortalWrt 360 V6 增强版 (Enhanced)

> **精简，是为了更持久地稳定运行。**

本固件基于 **ImmortalWrt SNAPSHOT** 编译，专为 360 V6 (IPQ60xx) 打造。在维持系统轻量化的同时，集成了最常用的功能，实现开箱即用。

## 🧩 核心功能
为了兼顾性能与 WiFi 稳定性，仅精选集成了以下 **5 款** 插件：
- **可视化管理**：`luci-app-opkg` (图形化软件安装)
- **文件管理**：`luci-app-fileassistant` (管理备份与传输)
- **远程办公**：`luci-app-wol` (网络唤醒)
- **自动维护**：`luci-app-autoreboot` (定时重启)
- **广告屏蔽**：`luci-app-adguardhome` (DNS 过滤)

## 📥 登录信息
- **管理地址**：`192.168.2.1`
- **用户名**：`root`
- **初始密码**：无 (`none`)

## ⚠️ 注意事项
1. **AdGuardHome**：首次使用需进入「服务」面板完成初始化，建议上游 DNS 设置为 `223.5.5.5`。
2. **不再加码**：本版本已达到 360 V6 的性能平衡点，后续将不再增加更多功能。
